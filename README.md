# 🎓 AI Powered Placement Prediction System

A Machine Learning project that predicts whether a student is likely to get placed based on academic and extracurricular factors.

## Features

- Logistic Regression Classification
- Streamlit Web Application
- Placement Probability Prediction
- ROC Curve Analysis
- Hyperparameter Tuning using GridSearchCV

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-Learn
- Streamlit
- Joblib

## Model Performance

- Accuracy: 80.85%
- AUC Score: 0.8837

## Input Features

- CGPA
- Internships
- Projects
- Workshops/Certifications
- Aptitude Test Score
- Soft Skills Rating
- Extracurricular Activities
- Placement Training
- SSC Marks
- HSC Marks

## How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
```
